# Автоматизация выгрузки товаров Wildberries

Скрипт для автоматического экспорта товаров из Wildberries API в формат, готовый для вставки в Google Sheets.

## Описание

Программа подключается к API Wildberries, получает список товаров по указанному objectId, обрабатывает данные и автоматически копирует их в буфер обмена в формате TSV (табуляция), который Google Sheets автоматически распознает и размещает по колонкам.

## Требования

- Python 3.7+
- Токен доступа к API Wildberries

## Установка

1. Клонируйте репозиторий или скачайте файлы проекта

2. Создайте виртуальное окружение (рекомендуется):
```bash
python -m venv venv
```

3. Активируйте виртуальное окружение:
   - Windows: `venv\Scripts\activate`
   - Linux/Mac: `source venv/bin/activate`

4. Установите зависимости:
```bash
pip install -r requirements.txt
```

## Настройка

1. Создайте файл `.env` в корне проекта

2. Добавьте в него ваш токен API Wildberries:
```
WB_API_TOKEN=ваш_токен_здесь
```

## Запуск

```bash
python main.py
```

После выполнения скрипт:
- Проверит подключение к API Wildberries
- Получит товары по objectId (по умолчанию: 435)
- Обработает данные и выведет их в консоль
- Автоматически скопирует данные в буфер обмена

Данные можно сразу вставить в Google Sheets (Ctrl+V).

## Изменение objectId

Для получения товаров из другой категории измените значение `objectId` в строке 114 файла `main.py`:
```python
products = get_wb_products(435)  # Замените 435 на нужный objectId
```
